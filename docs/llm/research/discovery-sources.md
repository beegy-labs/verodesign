## Search Query Templates

When using `.add/pattern-research.md`, prefer these query shapes:

| Shape | Example |
| ----- | ------- |
| Year + topic | `2026 design tokens architecture` |
| Spec status | `W3C DTCG stable spec changes` |
| Tier 1 source + topic | `Vercel Geist token architecture` |
| Vs. comparison | `oklch vs hsl token authoring 2026` |
| Site filter | `site:adobe.com spectrum tokens` |

## Anti-Sources (do NOT base decisions on)

| Source | Why not |
| ------ | ------- |
| Old (>2 years) blog tutorials | Likely Tailwind v3 or pre-DTCG |
| StackOverflow answers >3 years old | Likely outdated CSS |
| LLM-generated articles without citations | Hallucination risk |
| Influencer "tier list" videos | Subjective, low information density |
